Java 8 Functional Interface

什么是函数式接口

  • 只包含一个抽象方法的接口，称为函数式接口
  • 可以通过Lambda表达式来创建该接口的对象
  • 可以在任意函数式接口上使用 @FunctionalInterface 注解，这样做可以检查它是否是一个函数式接口

自定义函数式接口

作为参数传递Lambda表达式

为了将Lambda表达式作为参数传递，接收 Lambda 表达式的参数类型应该是与该 Lambda 表达式兼容的函数式接口的类型
